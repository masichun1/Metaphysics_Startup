## 日常操作速查

### 1. 启动网站

```powershell
cd "d:\马思纯\Metaphysics_Startup\Cross_Border_Project\website"
..\.venv\Scripts\python.exe app.py
```

启动后访问：
- **前台**: http://127.0.0.1:5000
- **后台**: http://127.0.0.1:5000/admin
- **后台密码**: `mystic2026`

---

### 2. 上架新产品

**方法一：用管理后台（最方便）**

1. 打开 http://127.0.0.1:5000/admin → Products → "+ New Product"
2. 填写英文标题、价格、SKU、产品描述(HTML)、SEO 关键词
3. 点 Save

**方法二：CSV 批量 + AI 生成（大批量时用）**

Step 1 — 编辑 CSV 文件:
```
文件位置: data/product_imports/real_products.csv
格式: sku,source_title,category,price,compare_at_price,cost,vendor,product_type,tags,materials,features,use_case,weight,image_urls,inventory_quantity
```

Step 2 — 运行 AI 生成:
```powershell
cd "d:\马思纯\Metaphysics_Startup\Cross_Border_Project"
.venv\Scripts\python.exe main.py run skill_01 --dry-run --csv-path "data/product_imports/你的文件.csv"
```

Step 3 — 导出会自动保存到 `data/exports/` 目录

**方法三：直接编辑 products.json**
```
文件位置: website/data/products.json
直接 JSON 添加新产品对象即可，刷新网站立即生效
```

---

### 3. 订单管理

1. 客户在前台下单 → 订单自动保存到 `website/data/orders.json`
2. 打开后台 http://127.0.0.1:5000/admin/orders 查看所有订单
3. 状态管理：Pending → Shipped → Completed，在后台直接改
4. 客户下单后会自动收到一封确认邮件（需要配置 SMTP）

> **注意**: 目前需要连接真实 SMTP 发件服务器才能自动发邮件。详见下方"配置邮件通知"。

---

### 4. 博客发布

1. 后台 → Blog → "+ New Post"
2. 填写 Slug（URL 地址，用英文连字符，如 `my-new-article`）
3. 填写标题（英文）、分类、日期
4. 正文用 HTML 格式写入（支持 h2, h3, p, ul, li, img, a 等标签）
5. 点 Save → 立即可在前台 /blog 看到

**AI 自动生成博客**（需要 API Key）:
```powershell
cd "d:\马思纯\Metaphysics_Startup\Cross_Border_Project"
.venv\Scripts\python.exe main.py run skill_03 --dry-run --topic "你的话题"
# 去掉 --dry-run 直接发布到网站
```

---

### 5. 客户联系的方式

**方式一：网站 Contact 页面** — 用户填表，信息保存在本地（目前需要查看表单数据）

**方式二：订单确认邮件** — 配置 SMTP 后自动发送。配置方法：
编辑 `website/data/settings.json`，添加：
```json
"smtp": {
  "host": "smtp.gmail.com",
  "port": 587,
  "user": "your-email@gmail.com",
  "password": "your-app-password",
  "from_email": "your-email@gmail.com",
  "from_name": "Mystic Sanctuary"
}
```

**方式三：Skill 04 弃单挽回** — 调用跨境项目自动化脚本，检测弃单并发送挽回邮件

---

### 6. 网站部署（让全世界用户都能访问）

**最简单的免费方案 — Cloudflare Tunnel:**

```powershell
# 1. 下载 cloudflared (一次)
winget install cloudflare.cloudflared

# 2. 启动网站后，另开一个终端：
cloudflared tunnel --url http://localhost:5000

# 3. 你会得到一个公网域名，如 https://mystic-sanctuary.trycloudflare.com
```

**更专业的方案 — Vercel / Railway / Render:**
将 website/ 目录推送到 GitHub，连接到上述平台一键部署。免费额度足够起步。

## 高级自动化

你之前搭建的 5 个 Skills 都在此目录下：

| Skill | 命令 | 功能 |
|-------|------|------|
| **01** 产品上架 | `python main.py run skill_01 --csv-path xxx.csv` | CSV → AI 生成英文内容 → 导出 |
| **02** 产品评论 | `python main.py run skill_02 --dry-run` | AI 生成真实感英文 Review |
| **03** 博客撰写 | `python main.py run skill_03 --topic "话题"` | AI 生成 SEO 博客文章 |
| **04** 客户通知 | `python main.py run skill_04 --dry-run` | 弃单检测 + 挽回邮件 |
| **05** 收益计算 | `python main.py run skill_05 --report-type weekly` | 利润报表(CSV+Slack) |

运行全部: `python main.py run-all --dry-run`

## 常见问题

**Q: 网站启动报错 "ModuleNotFoundError"?**
```powershell
cd "d:\马思纯\Metaphysics_Startup\Cross_Border_Project"
.venv\Scripts\python.exe -m pip install flask
```

**Q: 忘记后台密码?**
默认密码: `mystic2026`。修改: 编辑 `website/app.py` 搜索 `mystic2026` 改成新密码。

**Q: 怎么换产品图片?**
将图片放到 `website/static/images/` 目录，图片名与 SKU 一致（小写），如 `tjw-001.jpg`。然后在后台编辑产品确保 `image` 字段路径正确。

**Q: 怎么用 DeepSeek API 生成内容?**
确认 `.env` 文件中有:
```
ANTHROPIC_API_KEY=sk-你的deepseek-key
ANTHROPIC_DEFAULT_MODEL=deepseek-chat
```
