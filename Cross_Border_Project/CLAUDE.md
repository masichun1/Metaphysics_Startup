# Cross_Border_Project — 跨境业务主控

## 核心目标
打造**全英文环境**的海外玄学电商与全平台数字人社媒分发矩阵。

## 业务组成
- **电商端**：Shopify 独立站，面向欧美用户销售玄学相关产品（水晶、塔罗牌、风水摆件、灵性饰品等）。
- **社媒端**：HeyGen 数字人视频矩阵，覆盖 Snapchat / X / Instagram / Facebook / TikTok Lite / YouTube，统一引流至独立站。

## 内容语言标准
所有面向用户的前端展示内容、博客文章、产品描述、社媒文案、视频脚本等，必须是**地道、专业的全英文**，符合海外受众文化习惯。

## 目标人群
欧美 40+ 女性，对灵性、占星、塔罗、风水、冥想等领域有浓厚兴趣。

## 技术要求
- 对接 Shopify REST/GraphQL API 实现自动化运营。
- 对接 HeyGen API 或通过人工中转实现数字人视频的脚本供给。
- 所有自动化脚本需具备错误重试、日志记录和异常告警机制。
